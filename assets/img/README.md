# Imagens da LP

Coloque os arquivos **exatamente com estes nomes** nesta pasta (`assets/img/`).
Enquanto o arquivo não existir, o site mostra um placeholder — nada quebra.

| Arquivo               | Onde aparece                                  | Formato / dica |
|-----------------------|-----------------------------------------------|----------------|
| `logo-oralsin.png`    | Logotipo no header e no rodapé                | PNG (fundo transparente ou o badge verde). Altura exibida ~42px. |
| `fachada-oralsin.jpg` | Fundo da Seção 1 (hero), em baixa opacidade   | JPG landscape (largura ~2000px). Usada com opacidade 7%. |
| `dra-oralsin.jpg`     | Foto principal da Seção 1 (ao lado do texto)  | JPG retrato (proporção ~3:4, ex.: 900×1200). |

## Como adicionar os arquivos

**Opção A — pelo GitHub (mais fácil):**
1. Abra o repositório no GitHub, na branch `claude/lucid-bohr-n1ztz6`.
2. Entre na pasta `assets/img/` e clique em **Add file → Upload files**.
3. Arraste os 3 arquivos com os nomes acima e confirme o commit.
4. A Vercel republica sozinha e as imagens aparecem.
